# Student Management System

A simple JavaFX desktop application that performs CRUD operations on student data using JDBC and MySQL.

## Technologies
- Java 25
- JavaFX 25.0.2
- MySQL
- JDBC (MySQL Connector/J 9.6.0)

## Features
- Add student
- View students
- Delete student
- update student

## How to Run
Compile and run using JavaFX module path and MySQL JDBC driver.
